# AngularJSAndRequireJS


本项目是基于AngularJS + RequiresJS 搭建的前端模块化框架。

项目中充分体现了模块化开发、MVVM、IOC、Route等思想

项目框架设计：
    本框架有本人独立设计，命名为Horse，随着UI的逐步扩展，框架也逐渐健壮起来，
但是原则是一切从轻量考虑，所以有的地方还是有优化空间的；

目标：<br/>
    （1）让前端开发面向对象化；<br/>
    （2）让前端开发更有层次；<br/>
    （3）最终实现多版本、轻量、方便使用<br/>
